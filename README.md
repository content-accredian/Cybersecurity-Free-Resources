# Cybersecurity Free Resources

Welcome to the **Cybersecurity Free Resources** repository by [Accredian](https://www.accredian.com).[page:3]  
This repo curates high-quality, free learning materials to help students, professionals, and teams build strong foundations in cybersecurity, cloud security, and secure-by-design practices.[page:3]

---

## About this repository

This repository is designed as a one-stop hub for cybersecurity learning assets, including:

- Cheatsheets on core security concepts and architectures  
- PPT decks for training and awareness sessions  
- Practical references for modern security models and DevSecOps[page:3]

All resources are free to access and can be used for self-study, internal workshops, or organization-wide security awareness.

---

## Files in this repository

Currently, the repository includes resources such as:[page:3]

- `Encryption_Basics_Cheatsheet-3.docx` – A concise guide to symmetric, asymmetric, and hashing fundamentals, plus key management basics.[page:3]  
- `Zero_Trust_Model_Cheatsheet.docx` – Quick reference on zero trust principles, components, and implementation pillars.[page:3]  
- `cheatsheet_CloudSecurity_DevSecOps.docx` – Core ideas for securing cloud-native workloads and integrating security into CI/CD.[page:3]  
- `Human Error Cybersecurity Threat.pptx` – Training deck focused on human factors, social engineering, and security hygiene.[page:3]  
- `Ransomware Business Model of Cybercrime.pptx` – Presentation explaining how ransomware operations work and how organizations can defend against them.[page:3]

You can click on any file in the repository to preview or download it.

---

## How to use these resources

- Use the **cheatsheets** as quick references while designing or reviewing security controls.  
- Run **PPT sessions** as part of security awareness programs or internal brown-bag talks.  
- Integrate these materials into onboarding for engineering, DevOps, and business teams.  
- Combine multiple documents (for example, Zero Trust + Cloud Security + Human Error decks) to create a holistic security learning track.

If you are new to GitHub, click on a file and use the **Download** button to save it locally.
